# gcc -o {바꿀 이름} {파일 이름}

- 옵션으로 실행하는 이름 바꿔줌

ls 보면 컴파일된 파일은 앞에 a.out가 있음, 그리고 이름을 바꿀 수 있음

- 실행하는 방법 : **./{바꿔준 이름} or ./a.out**

# 파일 이름 바꾸기 & 삭제

- **mv {기존 이름} {바꿀 이름}**
- **rmdir {파일 이름}** : 단 파일 내에 내용이 없으면 삭제됨
