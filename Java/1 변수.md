# 변수

하나의 **값**을 **저장**할 수 있는 메모리 **공간**

# 변수 명명 규칙

- 첫 번째 글자는 문자,  $, _여야 하고 숫자로 시작할 수 없음
- 소문자와 대문자가 구분함 (카멜 표기법)
- 예약어 사용 금지, 즉 키워드 사용 금지

<aside>
💡 클래스는 파스칼 표기법을 사용 (EX) BackgroundColor)

</aside>

카멜, 파스칼, 헝가리안, 스네이크 표기법 등이 있음

**long : 30000L**

**float : 3.12345tF 알아두셔**
